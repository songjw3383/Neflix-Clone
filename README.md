# NETFLIX-Clone
## 목적
포트폴리오 목적으로 제작된 넷플릭스 클론 프로젝트입니다.
TMDB 의 API를 사용하여 영화 및 드라마의 정보를 가져온 후, Single Page Application 구조 형태로 제작되었습니다.
ReactJs 와 Axios를 사용해 데이터를 요청하는데 중심을 두고 제작하였으며, 그 외 간단한 CSS를 프로젝트에 적용했습니다.
